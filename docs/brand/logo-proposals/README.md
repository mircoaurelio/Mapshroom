# Mapshroom logo proposals

Generated as vector-friendly bitmap explorations with the built-in image generation tool.
The current `logo.ico` pixel mushroom was used only as a brand-history reference.

## Shared constraints

- App icon and favicon for a projection-mapping / visual-effects product
- Single centered mark with a strong silhouette
- Legible at 16 px and 32 px
- Coral red `#F04444`, near-black `#111114`, warm ivory `#F5F1E8`
- Flat geometry, no text, gradients, 3D, shadows, glow, watermark, or fine detail

## A — Heritage Pixel

Create a modern pixel/vector hybrid that preserves the recognizable red mushroom.
Use a broad cap, thick ivory stem, exactly three large cap spots, crisp geometric
edges, and fewer, larger pixels than the existing icon.

## B — Projection Surface

Merge a mushroom cap with a mapped projection surface. Use a broad red geometric
dome with bold ivory contour gaps following the cap curvature and a short black
trapezoid stem. It should feel like professional VJ/projection software.

## C — Mapping Nodes

Construct the mushroom from projection-mapping geometry. Use a broad black cap
outline, red inner surface, four large ivory control nodes connected as a simple
editable mesh, and a bold ivory stem. It must read first as a mushroom and second
as a mapping tool.

## Recommended next pass

Start from B for the production icon. Add one small asymmetric brand-specific cut
or notch to make the silhouette more ownable, then draw it as SVG and create
separately tuned 16 px, 32 px, and 512 px exports.
